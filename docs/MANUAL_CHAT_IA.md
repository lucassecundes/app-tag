# Manual de Utilização - Chat IA Tag Nativo

Bem-vindo ao novo Assistente Virtual Inteligente do Tag Nativo! Este manual explicará como utilizar as novas funcionalidades de inteligência artificial para gerenciar seus dispositivos de rastreamento de forma mais eficiente.

## 1. Acesso ao Chat IA
O acesso ao chat é feito através da **Tela de Dispositivos**.
- Procure pelo ícone flutuante roxo com um robô/bot, localizado acima do botão de adicionar dispositivo (+).
- Toque no ícone para abrir a interface de chat.

> **Nota:** Esta funcionalidade é exclusiva para assinantes **Premium**. Caso você não seja assinante, será solicitado que faça o upgrade para utilizar o recurso.

## 2. O Que Você Pode Fazer?
O Assistente IA está integrado diretamente com seus dispositivos e pode responder perguntas como:

### Localização
- "Onde está meu carro?"
- "Qual a localização da moto?"
- "Mostre a posição do caminhão."

### Listagem de Dispositivos
- "Quais dispositivos eu tenho?"
- "Liste minhas tags ativas."
- "Quantos dispositivos estão online?"

### Histórico e Endereços (Em breve)
- "Quais os lugares que eu mais visitei?"
- "Por onde meu carro andou hoje?"

## 3. Como Usar
1. **Digite sua dúvida**: Use o campo de texto na parte inferior da tela para digitar sua pergunta em linguagem natural.
2. **Prompts Rápidos**: Acima do campo de texto, você verá sugestões de perguntas comuns. Basta tocar em uma delas para enviar.
3. **Respostas**: O assistente processará sua solicitação e, se necessário, consultará o banco de dados para trazer informações precisas sobre seus dispositivos.

## 4. Segurança e Privacidade
- **Dados Seguros**: Todas as conversas são armazenadas de forma segura e apenas você tem acesso ao seu histórico.
- **Permissões**: O assistente só pode acessar os dispositivos vinculados à sua conta. Ele não tem acesso a dados de outros usuários.
- **Limites**: Para garantir a estabilidade do sistema, existe um limite de mensagens por minuto. Se você enviar muitas mensagens rapidamente, o assistente pedirá para aguardar alguns instantes.

## 5. Solução de Problemas
- **Erro de Conexão**: Verifique sua internet.
- **Dispositivo não encontrado**: Certifique-se de que o nome do dispositivo na pergunta corresponde ao nome cadastrado no app (ex: "carro" vs "Fiat Uno").
- **Funcionalidade bloqueada**: Verifique se sua assinatura Premium está ativa.

Para mais ajuda, entre em contato com o suporte através do menu de configurações.
